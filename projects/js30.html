Javascript 30

Javascript 30 is a thirty day challenge created by Wes Bos.

The challenge is to create thirty mini projects using vanilla JS in thirty days!

Wes gives an HTML and CSS template to start with and then it's up to me to finish each project.

This was a great first project. It is a simple design and it was straightforward to work with.

Take aways from this project:

Day 2 - (1/16/23) - JS Clock

github repo

go to project

Wes gave the clock design and it was my job to make the hands move according to the time. I am learning a lot by looking at his HTML and CSS and seeing what properties he used to accomplish this clean look. This project was pretty light on the JS and relied mostly on CSS.

Take aways from this project:

  • Practice working with the Date constructor and the methods available to it.
  • Used setInterval to refresh the clock every second.
  • Learned about the transition-timing-function property and how to control the speed of an animation or transition.
  • Learned how to adjust different cubic-bezier values in Chrome Developer Tools to get the desired custom values. This is how the seconds hand animates each second but slightly comes back to its previous position with each tick.

My additions to this project:

  • I added the current date to the clock. Used a z-index so that the hands don't go "over" the date as they cycle through the element.

Day 3 - (1/17/23) - CSS Variables & JS

github repo

go to project

Wes gave the starting design and it was my job to get the form input values with Javascript and make the website adjust to the actions specified by the user. The challenge focused on using CSS variables with Javascript. I learned a lot from reading the HTML and CSS that Wes set up for this project.

Take aways from this project:

  • I knew about the range form input but I didn't know about the color form input that pulls up a color wheel. It's so cool that the functionality comes out of the box with HTML. I also reviewed some other form input types to get inspired for what I can use on future projects.
  • This project taught me how to use CSS variables. First, you declare the variable at the highest element in the CSS file which is :root. Variable names start with a double dash (for example --color: #fff). When you want to use a variable on a CSS property, you reference the variable with the var keyword (for example color: var(--color). Full documentation is available here.
  • I didn't know too much about the root element of the DOM before this project. I learned that you can access it with document.documentElement in Javascript and :root in CSS as mentioned above.

My additions to this project:

  • I could have added more elements to alter the image.
  • I made the design mobile responsive by adding the object-fit: cover property on the image selector and adjusted some of the spacing issues.
